
# Kalman Filter — 1D Position Tracker (Streamlit)

Interactive demo: tune process/measurement noise and see how the Kalman estimate balances model vs. measurements.

## Run locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy (Streamlit Community Cloud)
1. Push this folder to a public GitHub repo.
2. Go to https://share.streamlit.io , sign in with GitHub, and create a new app.
3. Select your repo and choose `streamlit_app.py` as the app file.
4. Deploy — share the URL in your LinkedIn post.
